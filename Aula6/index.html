Flex Box

São caixas flexíveis que podem se adaptar aos breakpoints declarados no CSS

Elementos e atributos trabalhados junto com o Flex Box:

      Display: exibe um elemento
      Flex-direction: direção dos elementos, por padrão declaramos em row (linha)
      Flex-wrap: define quem quebra linha da caixa
      Flex-flow: atalho para o uso do direction e wrap
      Justify-content:conteúdo justificado
      align-intems: alinhamento de itens dentro do elemento
